# 台灣景氣指標研究

## 資料來源
國家發展委員會（國發會）景氣指標查詢系統
- 官方網站：https://index.ndc.gov.tw/
- API/資料下載：https://index.ndc.gov.tw/n/zh_tw/data/eco

## 台灣景氣指標架構

### 領先指標（Leading Indicators）
領先指標具有領先景氣變動性質，可預測未來景氣走向，通常有 6-9 個月的領先時間。

**構成項目（7項）：**
1. 外銷訂單動向指數（以家數計）
2. 製造業營業氣候測驗點
3. 股價指數
4. 實質貨幣總計數 M1B
5. 實質半導體設備進口值
6. 建築物開工樓地板面積
7. 工業及服務業受僱員工淨進入率

### 同時指標（Coincident Indicators）
同時指標與景氣變動同步，代表目前經濟狀況。

**構成項目（待查）：**
- 工業生產指數
- 製造業銷售量指數
- 批發、零售及餐飲業營業額
- 非農業部門就業人數
- 實質海關出口值
- 實質機械及電機設備進口值
- 製造業單位產出勞動成本指數

### 落後指標（Lagging Indicators）
落後指標變動落後於景氣變動，用於確認景氣循環。

**構成項目（待查）：**
- 失業率
- 製造業單位產出勞動成本指數
- 全體金融機構放款與投資
- 工業及服務業經常性受僱員工人數
- 製造業存貨率

## 政府開放資料平台調查結果

**資料集：景氣指標及燈號**
- 網址：https://data.nat.gov.tw/dataset/6099
- 提供機關：國家發展委員會
- 更新頻率：每月
- 授權方式：政府資料開放授權條款-第1版（免費）
- 資料格式：ZIP（內含多個 CSV 檔案）
- 資料內容：包含領先、同時、落後指標及景氣對策信號

**資料欄位：**
- DATE
- 領先指標綜合指數、領先指標不含趨勢指數
- 同時指標綜合指數、同時指標不含趨勢指數
- 落後指標綜合指數、落後指標不含趨勢指數
- 景氣對策信號綜合分數、景氣對策信號
- 各項構成項目原始值（外銷訂單、M1B、股價指數、工業生產等）

**限制：**
- 沒有提供 REST API，只有檔案下載
- 每月更新時會回溯修正歷史資料
- 需要定期下載 ZIP 檔案並解壓縮

## 決定採用的方案

由於國發會沒有提供 REST API，我們將採用以下方案：

### 方案：定期下載政府開放資料

**實作方式：**
1. 建立 Taiwan Fetcher，定期下載國發會開放資料 ZIP 檔案
2. 解壓縮並解析 CSV 檔案
3. 將資料寫入資料庫

**優點：**
- 官方權威資料來源
- 免費且穩定
- 資料完整涵蓋所有景氣指標

**缺點：**
- 需要處理檔案下載和解壓縮
- 沒有 API，需要模擬瀏覽器下載或直接 HTTP 請求
